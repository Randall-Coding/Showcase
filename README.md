Running Rails 5.2. &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; Ruby version 2.5.  &nbsp;&nbsp;&nbsp; Rspec 3.8/Capybara 3.12 for testing.  &nbsp;&nbsp;&nbsp; PostgreSQL database. &nbsp;&nbsp;&nbsp; <br>  Deployed on Heroku

## Tests
